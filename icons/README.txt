# Icon Placeholders

Untuk production, ganti file ini dengan icon PNG yang sebenarnya.

Ukuran yang dibutuhkan:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

Sementara ini, extension akan tetap berfungsi tanpa icon (Chrome akan gunakan default icon).

Untuk membuat icon cepat, bisa gunakan:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- Atau design sendiri di Figma/Photoshop

Warna yang disarankan:
- Background: #0a0f19 (dark blue)
- Accent: #00f2ff (cyan)
- Logo: Huruf "T" atau simbol chess piece
